# README

## 修改字体

该插件会修改当前页面的字体。

每一个字符按如下的顺序，如果在前一个没匹配到，就会使用后一个字体显示。

``` text
'Cascadia Code', Consolas, '霞鹜文楷', '仓耳今楷01'
```

## 控制栏

位于窗口左上角，采用仿 macOS 的红绿灯样式设计。

- **红色（关闭）**
  - 点击后，浮窗以自身正中心为基准点，向中心点等比缩小，同时透明度逐渐增加，直至完全不可见。
- **黄色（最小化）**
  - 点击后，浮窗向右下角平移，同时向中心点等比缩小并增加透明度，直至完全不可见。
  - 浮窗消失后，屏幕右下角会出现一个恢复按钮。点击该按钮，浮窗将以完全相反的过程（从右下角移回原位、从中心点等比放大、透明度降低）恢复至原始状态。
- **绿色（最大化 / 还原）**
  - **常态下点击**：浮窗向正中心等比放大，同时浮窗中心移动至与屏幕窗口中心重合。
  - **最大化状态下点击**：浮窗向正中心等比缩小，同时浮窗中心移动回原始位置。

## 标题栏

`标题栏` 中间区域支持鼠标按下拖拽以移动浮窗位置。该区域中央显示当前窗口标题。

## 课程描述输入框

用于输入课程描述文本。输入内容的前后空白符将被自动忽略。

**输入格式说明：**

- 输入内容应包含 **课程名称** 与可选的 **课程节次信息**。
- **课程节次信息** 可通过 `第x节` 格式指定本周第几节课，或直接描述为星期几的课程。

!!! attention N.B.
    - 形如 `第x节` 的文本 **仅** 会被识别为 **本周的第 x 次课**，不会被识别为当天的节次编号。
      - *例如：输入 `周一晚上第二节高数` 将仅匹配“本周第二次高等数学课”，而非“周一晚上的第二节课”。*
    - 周次信息将被忽略，系统始终以 **当前周次** 为准进行匹配，即使输入中指定了特定周次。

课程名称支持使用常见简写，且 **不区分大小写**。当前支持的课程别名映射列表见文档末。

### 快捷键行为

- 若 `输入框` 内容 **为空**，按下 <kbd>Ctrl</kbd> + <kbd>Enter</kbd> 将触发一次 **课程随机抽取**。
- 若 `输入框` 内容 **不为空**，按下 <kbd>Ctrl</kbd> + <kbd>Enter</kbd> 将触发一次 **提交**。
- 按下 <kbd>Tab</kbd> 键可将焦点切换至 **反馈内容输入框**。

## 反馈内容输入框

- 当窗口处于 **最大化** 状态时，此输入框将自适应扩展至最大可用空间。
- **快捷键行为**：
  - 若 **`课程描述输入框` 为空**，在此处按下 <kbd>Ctrl</kbd> + <kbd>Enter</kbd> 将触发一次 **课程随机抽取**。
  - 若 **`课程描述输入框` 不为空**，在此处按下 <kbd>Ctrl</kbd> + <kbd>Enter</kbd> 将触发一次 **提交**。

## 随机抽取课程逻辑

### 抽取流程

1. **筛选可抽取课程**：基于当前教学周次与课程的 `教学时间区间` 进行匹配。
2. **获取有效节次**：在筛选出的课程中，进一步找出当前周次实际安排的所有课程节次。
3. **等概率随机抽取**：从上述所有有效课程节次中，以等概率随机抽取一条结果。
4. 将抽取结果填入 `课程描述输入框` 。

抽取保证不重复。具体来说，在所有课程都抽取完成前，不会出现课程重复；当所有课程都已经被抽取过一次后，将所有课程都视为从未抽取过，重复之前的流程。

### 可抽取课程判定规则

课程是否可被抽取取决于其 `教学时间区间` 是否包含 **当前周次**。

- *例：当前为第 7 周，某课程 `教学时间区间` 为 1-12 周，则该课程 **被选中**。*
- *例：当前为第 7 周，“形势与政策”课程 `教学时间区间` 为 13-16 周，F则该课程 **不被选中**。*

!!! attention N.B.
    `教学时间区间` 判定 **不涉及单双周逻辑**。
    例如：某课程安排在第 1-12 周的 **双周** 进行，若当前处于第 7 周（单周），该课程依然会作为候选课程参与第一步筛选。最终的 **单双周过滤** 发生在第二步 `获取可抽取课程节次` 阶段。

### 可抽取课程命名规则

将把各课程别名列表的首项作为其显示名称。如果某门课程在 `教学时间区间` 中有多节，则在前面补上星期信息。例如：高数会显示为 `周一高数` 和 `周三高数`。

!!! attention N.B.
    是否显示星期信息与单双周无关。例如 `数字逻辑` 在单周有两节，双周一节。那么不论单双周，都将在其前面补上星期信息。

## 随机抽取课程按钮

位于 `课程描述输入框` 的右侧，按钮高度与输入框保持一致。

## 填写并复制按钮

- **启用条件**：仅当 **`课程描述输入框` 内容不为空** 时，此按钮才处于可点击状态。
- **禁用状态**：若 `课程描述输入框` 为空，按钮将呈禁用样式。
  
点击后，脚本将自动根据 `课程描述输入框` 和 `反馈内容输入框` 的内容，自动填写表单，并将生成的反馈消息复制到用户剪切板中。

表单填写完成后，脚本自动将页面滑到最底部。
